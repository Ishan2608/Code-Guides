00 — Foundations
What is a Discrete RV?

A random variable X is discrete when it can only take a countable number of distinct values — usually whole numbers. Think counts: number of heads, number of arrivals, number of defects.

PMF — Probability Mass Function

The PMF gives the exact probability that X equals a specific value k. It answers: "What is the chance of getting exactly k?"

P(X = k) = f(k), k ∈ support
Two PMF Rules (always hold)

Every valid PMF must satisfy these two axioms — they are non-negotiable:

① P(X=k) ≥ 0 for all k
② Σ P(X=k) = 1 (sums to one)
01 — Bernoulli
Ber(p) Bernoulli Distribution The Coin Flip
Simple Intuition

"One trial. Two outcomes. Either it happens — with probability p — or it doesn't."

Technical Definition

The simplest building block of all discrete distributions. X takes value 1 (success) with probability p, and 0 (failure) with probability 1−p. Every other discrete distribution is built from or related to Bernoulli trials.

Conditions
  • ▸Exactly one trial
  • ▸Exactly two outcomes: success or failure
  • ▸P(success) = p is fixed and known
Real-World Use Cases

A patient either responds to a drug or not. A manufactured part is defective or not. An email is spam or not. A student passes or fails a single exam.

PMF
P(X=k) = pk(1−p)1−k
         k ∈ {0, 1}
// Equivalently: P(X=1)=p, P(X=0)=1−p

Key insight: Variance is maximized at p = 0.5 (most uncertain) and is zero at p = 0 or p = 1 (certainty).

02 — Binomial
B(n,p) Binomial Distribution Repeated Coin Flips
Simple Intuition

"You flip a coin n times. How many times does it land heads? That count follows a Binomial distribution."

Technical Definition

The sum of n independent, identical Bernoulli(p) trials. X counts the total number of successes. The C(n,k) term accounts for all the different orderings in which k successes can occur among n trials.

Conditions (BINS)
  • ▸Binary outcomes only
  • ▸Independent trials
  • ▸Number of trials n is fixed
  • ▸Same probability p each trial
Real-World Use Cases

Quality control: how many defective items in a batch of 100? Drug trial: how many of 50 patients recover? Polling: how many of 1000 voters choose candidate A?

PMF
P(X=k) = C(n,k) · pk · (1−p)n−k
         k ∈ {0, 1, 2, …, n}
// C(n,k) = n! / [k!(n−k)!] — "n choose k"
Parameters & Moments

Approximation: When n is large and p is small (np < 10), use Poisson(λ=np). When n is large and np > 5, use Normal(μ=np, σ²=np(1−p)).

03 — Geometric
Geo(p) Geometric Distribution The Waiting Game
Simple Intuition

"You keep trying until you finally succeed. How many tries does it take? That's the Geometric distribution."

Technical Definition

X is the number of independent Bernoulli(p) trials needed to get the first success. Each failed attempt is (1−p), and the final trial succeeds with probability p. The PMF decays exponentially — the longer you wait, the less likely it becomes.

Conditions
  • ▸Trials are independent
  • ▸Each trial: same probability p of success
  • ▸Trial continues until first success
Real-World Use Cases

Number of calls until a customer answers. Number of job applications before getting an offer. Number of free-throw attempts until first miss. Server requests until first error.

⚡ Memoryless Property:
P(X > m+n | X > m) = P(X > n)
Past failures don't affect future probability.
PMF
P(X=k) = (1−p)k−1 · p
         k ∈ {1, 2, 3, …}
// k−1 failures, then 1 success

Caution — two conventions: Some textbooks define X as the number of failures before success (support starts at 0). Always check which convention is used; E[X] differs by 1.

04 — Negative Binomial
NB(r,p) Negative Binomial Distribution Geometric Generalized
Simple Intuition

"You don't stop after 1 success — you need r successes. How many total trials does that take? That's the Negative Binomial."

Technical Definition

X is the number of trials needed to achieve exactly r successes in independent Bernoulli(p) trials. The last trial must be the r-th success, and the preceding k−1 trials contain exactly r−1 successes. Geometric is the special case r = 1.

Conditions
  • ▸Independent Bernoulli trials, fixed p
  • ▸Fixed target: exactly r successes
  • ▸X counts total trials (not failures)
Real-World Use Cases

Recruiting: trials until 5 qualified candidates found. Clinical: patients screened until r respond to drug. Manufacturing: items inspected until r pass QC. Sales: calls made until r deals closed.

PMF
P(X=k) = C(k−1, r−1) · pr · (1−p)k−r
         k ∈ {r, r+1, r+2, …}
// k−r failures before the r-th success
// Last trial is always the r-th success
Parameters & Moments

Alternative parameterization: Some books define X as number of failures before r-th success (support starts at 0). Then E[X] = r(1−p)/p.

05 — Poisson
Pois(λ) Poisson Distribution Events in Time / Space
Simple Intuition

"Events happen randomly but at a known average rate. How many happen in a fixed window of time or space? That's Poisson."

Technical Definition

X counts the number of events occurring in a fixed interval, where events occur independently at a constant average rate λ (lambda). Derived as the limit of Binomial(n, p) as n→∞ and p→0 with λ = np held constant. The e−λ factor is the normalizing constant ensuring the PMF sums to 1.

Conditions (RICE)
  • ▸Randomness: events occur at random
  • ▸Independence: non-overlapping intervals independent
  • ▸Constant rate λ per unit
  • ▸Exclusivity: max 1 event per instant (no simultaneous)
Real-World Use Cases

Website hits per minute. Emails received per hour. Earthquakes per year in a region. Typos per page. Calls arriving at a call center. Radioactive decay events per second.

PMF
P(X=k) = e−λ · λk / k!
         k ∈ {0, 1, 2, 3, …}
// λ = average rate (events per unit)
// e ≈ 2.71828 (Euler's number)

Remarkable fact: Mean = Variance = λ. This is unique to Poisson. If you observe Var ≈ Mean in count data, Poisson is a natural starting model. Overdispersion (Var > Mean) suggests Negative Binomial instead.

06 — Hypergeometric
HG(N,K,n) Hypergeometric Distribution Sampling Without Replacement
Simple Intuition

"You have a bag of N marbles: K are red, the rest are blue. You draw n without putting any back. How many red marbles do you get?"

Technical Definition

Unlike Binomial (with replacement), Hypergeometric models sampling without replacement from a finite population. Each draw changes the composition of the remaining pool, so trials are not independent. The PMF uses combinations to count favorable selections over all possible selections.

Parameters
  • ▸N — total population size
  • ▸K — number of "successes" in population
  • ▸n — sample size drawn
  • ▸No replacement between draws
Real-World Use Cases

Lottery: drawing 6 numbers from 49. Acceptance sampling: testing 10 items from batch of 50 (5 defective). Card games: drawing hands from a finite deck. Medical: selecting patients from a group for treatment.

PMF
P(X=k) = C(K,k) · C(N−K, n−k)
            ─────────────────────
            C(N, n)
// max(0, n+K−N) ≤ k ≤ min(n, K)
Parameters & Moments
Mean E[X]
nK/N
Variance
n·(K/N)·(1−K/N)·(N−n)/(N−1)
Finite pop. correction
(N−n)/(N−1)
Approx. by
Binomial if n/N < 0.05
Key difference from Binomial: The finite population correction factor (N−n)/(N−1) reduces variance. When n is small relative to N (n/N < 0.05), the Hypergeometric ≈ Binomial(n, K/N).

07 — Discrete Uniform
DU(a,b) Discrete Uniform Distribution Equally Likely Outcomes
Simple Intuition

"Roll a fair die. Every face is equally likely. That's Discrete Uniform — all outcomes have identical probability."

Technical Definition

X takes each integer value in {a, a+1, …, b} with exactly the same probability 1/(b−a+1). It is the discrete analogue of the continuous Uniform distribution, and represents maximum entropy — complete uncertainty about which specific value will occur.

Conditions
  • ▸Finite number of outcomes
  • ▸Every outcome equally probable
  • ▸Support is a set of consecutive integers
Real-World Use Cases

Rolling a fair die (a=1, b=6). Drawing a random card rank. Random number generators. Lottery ticket numbers. Randomly selecting a day of the week.

PMF
P(X=k) = 1 / (b − a + 1)
         k ∈ {a, a+1, …, b}
// n = b−a+1 equally likely values
// Every bar in the PMF is the same height
Parameters & Moments
Parameters
a, b ∈ ℤ, a ≤ b
Support
k ∈ {a, …, b}
Mean E[X]
(a+b)/2
Variance
(n²−1)/12, n=b−a+1
Entropy
log(b−a+1) (maximum)
Skewness
0 (symmetric)
Maximum entropy: Among all distributions on n outcomes, the Discrete Uniform has the highest entropy — it represents the state of maximum uncertainty, encoding zero prior information about which outcome will occur.
